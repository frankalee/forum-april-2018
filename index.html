Google Lighthouse

Page optimization and auditing

What is Lightouse?

Google's open-source, automated tool for improving the quality of web pages.

A good baseline for improving web page performance, with a focus on mobile.

The "Audits" tab of the Chrome console.

Define "Quality"

Lighthouse defines quality in 5 ways.

  • Performance
  • PWA-ness
  • Accessibility
  • Best Practices
  • SEO

Define "Performance"

Load speed with a mobile focus. Chrome loads the page over a simulated 3G connection.

The biggest metrics are:

  • First meaningful paint - starts to look right
  • First interactive - usable
  • Consistently interactive - completely loaded

How do we improve performance?

Reduce the critical request chain!

  • Reduce page weight - optimize images, minify CSS & JS, enable compression
  • Lazy-load images when possible
  • Reduce render-blocking CSS & JS
